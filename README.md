# Notepad Windows Forms

A simple Notepad application developed using C# and Windows Forms.

## Features

* Create a new text file
* Open existing text files
* Save files
* Save As
* Cut, Copy, and Paste
* Select All
* Find text in the document
* Undo and Redo support
* User-friendly Windows Forms interface

## Technologies Used

* C#
* .NET Framework
* Windows Forms

## Project Structure

* `NotepadForm.cs` : Main application form
* `Tools.cs` : Undo/Redo management
* `ITools.cs` : Interface for editing tools
* `FindForm.cs` : Search functionality

## How to Run

1. Open the solution in Visual Studio.
2. Build the project.
3. Run the application using `F5` or the Start button.
